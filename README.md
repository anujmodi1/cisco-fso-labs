Install Jypyter Notebook
===========================
https://jupyter.org/install

Jupyter Notebook
Install the classic Jupyter Notebook with:
#pip install notebook
To run the notebook:
#jupyter notebook

Install GIT
================
https://github.com/git-guides/install-git

MAC and Linux:
#sudo apt-get install git-all
